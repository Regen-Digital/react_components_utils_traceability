# Single Page Application

## Prerequisites

- [npm](https://www.npmjs.com/) (>= 9.6.7)
- [yarn](https://yarnpkg.com/) (>= 1.22.19)

## Usage

```bash
# Install dependencies
yarn install
# Run the app
yarn start
# Build the app
yarn build
# Run tests
yarn test
# Run tests coverage
yarn coverage
# Start storybook
yarn storybook
```

## Cypress

- Run cypress

```
yarn run cypress:open

```

- Select E2E testing
- Choose Chrome as a browser
- Click on Start
